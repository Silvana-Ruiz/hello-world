# Cereal Recipe

1. First grab a bowl, milk, and cereal. The best cereals are:
    * _Lucky charms_
    * _Fruit loops_
2. Place the bowl on a flat surface.
3. Put the cereal in the bowl.
4. Pour the milk in.
